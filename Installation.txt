INSTALLION TUTORIAL:

1. Download XAMPP

In a code editor: 
2. Edit the $servername in includes/database.inc.php to match the localhost route.
3. Import the necessary sql file from the project folder "project3/sql/home.sql"

On a preferred browser
4. Go to localhost:(your localhost number)/phpmyadmin 
5. Open the application by going to localhost:(your localhost set up)/project3/index.php
